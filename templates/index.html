JSON API

  • GET /skus — list SKUs
  • GET /skus/{id} — get one SKU (includes full component structure)
  • POST /skus — create SKU (JSON body)
  • PUT /skus/{id} — update SKU
  • DELETE /skus/{id} — delete SKU

Composite SKUs include a components array of { "sku_id", "quantity" } objects. Simple SKUs must have an empty components array.
